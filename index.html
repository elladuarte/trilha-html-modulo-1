Falando sobre as tags e seus atributos

Nesta aula foi apresentada a tag i que tem por objetivo marcar o texto em itálico. Em seguida, a tag strong que permite colocar o texto ou um fragmento dele em negrito. Quando precisamos inserir uma imagem em um documento HTML,fazemos o uso da tag img. Para ajustar o tamanho da imagem, foram utilizados os atributos width e height, que correspondem à largura e à altura, respectivamente. Além disso, o atributo alt foi empregado para fornecer uma descrição da imagem. O atributo src é utilizado para colocar o endereço da imagem.

Entre outros atributos principais pertinentes as tags do HTML, temos como exemplo o atributo style que permite colocar comandos CSS direto no elemento. Por exemplo foi aplicado tal atributo na tag strong e este permitiu alterar a cor da palavra style para azul.

As tags h1, h2, h3, h4, h5 e h6 são utilizadas para definir subtítulos em uma página.

"A tag p define o corpo do texto, ou seja, seus parágrafos. E neste parágrafo foi utilizada a tag blockquote para definir uma citação."

A tag mark é utilizada para destacar trechos dos textos que são considerados relevantes. Para sublinhar um texto é importante usar a tag u.

Extra 1: Tag sub

A tag sub* é usada para colocar um texto em formato subscrito, como em fórmulas químicas. Pode ser usado por exemplo em fórmulas químicas como: O2*

Extra 2: Tag sup:

Já a tag sup* é usada para sobrescrito, sendo comum em potências matemáticas ou referências de rodapé. Exemplo: 24 = 16.

Diferença entre Listas Ordenadas e Não Ordenadas

As listas ordenadas são representadas pela tag ol, enquanto as listas não ordenadas são representadas pela tag ul.

"É importante observar que dentro dessas tags é obrigatório colocar a tag li porque essa tag de fato vai trazer os itens da lista que está sendo implementada dentro do documento."

Segue abaixo a tabela demonstrando como as listas ordenadas e não ordenadas vão se comportar no HTML:

Lista Ordenada Lista Não Ordenada
  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  • Item 1
  • Item 2
  • Item 3
  • Item 4

Tag Link

A tag a é utilizada para inserir um link para navegar entre páginas. Essa tag possui como atributo o target, sendo que este fornece as opções de _self (que é um comportamento padrão), porque quando clicado no link, ele abre este na mesma aba. Já o _blank abre o site em uma outra aba. O atributo title é utilizado para fornecer mais informações quando o usuário passa o mouse sobre determinado trecho.
